# A Simple Math Interpreter Written in Dart

## Features
- **Supported Operators**
    - `+`,`-`,`*`,`/`,`^`
- **Parentheses** *(Precedence)*
    - `(`,`)`
- **Lexer**
    - Generates `Tokens` from `Text`
- **Parser**
    - Generates `Abstract Syntax Tree (AST)` from `Tokens`
- **Interpreter**
    - Generates output by executing the `Abstract Syntax Tree (AST)`
- **Open Source**
